# 🎨 真实感与拍摄角度优化说明

## 📋 优化概览

根据用户反馈，我们对社交媒体营销图片生成器进行了重大优化，主要解决两个核心问题：
1. **AI感太重** - 减少过度美化，让图片更真实自然
2. **拍摄角度单一** - 增加多种高级运镜拍摄角度

---

## ✨ 核心优化内容

### 1. 系统提示词全面升级

#### 🎯 真实感核心原则
新增了严格的"真实感核心原则"，强制要求AI遵循：

```
✅ 避免过度美化
   - 不要过度美白、过度磨皮、过度液化
   - 保持人物和商品的真实质感

✅ 自然光线
   - 使用真实的环境光、自然光
   - 避免过亮、过假的光照效果

✅ 真实纹理
   - 保留皮肤的自然纹理
   - 保留商品的真实材质
   - 不要过度平滑

✅ 自然表情
   - 人物表情要自然、放松
   - 避免僵硬的假笑或过度夸张

✅ 真实场景
   - 场景要真实可信，有生活气息
   - 避免过于完美、人工感太强

✅ 适度后期
   - 像专业摄影师的适度后期
   - 而非AI生成的过度效果

✅ 真实细节
   - 保留一些真实的小细节
   - 头发丝、衣服褶皱、阴影等
```

#### 📷 拍摄角度多样化
新增了10种不同的拍摄角度，每次生成时自动选择，避免千篇一律：

| 拍摄角度 | 特点 | 适用场景 |
|---------|------|---------|
| 45度侧拍 | 最常用，立体有层次 | 通用场景 |
| 俯视/俯拍 | 突出商品展示 | 商品展示为主 |
| 仰视/仰拍 | 增强气势 | 高端品牌 |
| 侧拍/侧面 | 展现轮廓线条 | 人物展示 |
| 斜角/对角线 | 动感十足 | 运动场景 |
| 过肩镜头 | 增加代入感 | 互动场景 |
| 第一视角手持 | 亲切真实 | 日常分享 |
| 特写 | 突出细节 | 商品细节 |
| 中景 | 平衡展示 | 通用场景 |
| 远景 | 营造氛围 | 环境展示 |

#### 💡 镜头语言指导
新增专业的镜头语言指导：
- 景深控制：适当使用背景虚化，突出主体
- 焦点变化：可以在商品或人物上切换焦点
- 构图法则：三分法、黄金分割、引导线
- 动态模糊：适当增加真实感，避免过于锐利

#### ☀️ 光线质感优化
强调真实自然的光线效果：
- 自然光：窗边光、户外自然光、夕阳暖光
- 环境光：室内灯光、咖啡馆氛围光
- 柔和散射：避免强光直射
- 真实阴影：保留自然的阴影和高光
- 色温自然：不过度调整色温

### 2. 图片生成工具智能化

#### 🔧 自动增强功能
在 `src/tools/image_gen_tool.py` 中新增了 `enhance_prompt_for_realism` 函数，自动为提示词添加真实感增强描述：

```python
# 真实感增强关键词
REALISM_ENHANCERS = [
    "真实质感", "自然肤色", "适度后期", "保留纹理",
    "不过度美化", "自然过渡", "真实细节", "生活气息",
]

# 拍摄角度关键词
SHOT_ANGLES = [
    "45度侧拍构图", "自然俯拍视角", "微微仰拍角度", 
    "侧面轮廓展现", "斜角动态构图", "过肩镜头视角",
    "第一人称手持", "中景自然构图", "适当俯视突出商品",
    "立体侧拍", "自然角度", "真实拍摄视角",
]

# 光线质感关键词
LIGHTING_STYLES = [
    "自然窗边散射光", "温暖午后自然光", "柔和室内环境光",
    "真实环境光", "适度光影层次", "自然阴影",
    "不过度曝光", "真实色温",
]
```

每次生成时，工具会：
1. 随机选择2个真实感增强关键词
2. 随机选择1个拍摄角度
3. 随机选择1种光线风格
4. 自动添加到用户提示词中
5. 避免重复添加已有的关键词

## 🔄 对比效果

### 优化前：
```
❌ AI感过重
   - 过度美白，肤色不自然
   - 过度磨皮，失去皮肤纹理
   - 过度液化，人物比例失调
   - 光线过亮过假
   - 每次都是正面视角

❌ 拍摄角度单一
   - 几乎都是正面拍摄
   - 缺乏镜头语言
   - 构图单调乏味
   - 缺少立体感

❌ 细节不够真实
   - 看起来太"完美"
   - 缺少生活气息
   - 缺少真实小细节
```

### 优化后：
```
✅ 真实自然
   - 保留皮肤自然纹理
   - 保持人物真实质感
   - 适度后期，不假白
   - 光线自然真实
   - 有真实的生活气息

✅ 拍摄角度多样
   - 10种不同拍摄角度
   - 每次随机选择不同角度
   - 有镜头感和立体感
   - 构图更专业

✅ 细节真实可信
   - 保留头发丝、衣服褶皱
   - 真实的阴影和高光
   - 商品材质还原度高
   - 有真实的种草感
```
