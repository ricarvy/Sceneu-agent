📖 使用说明

  • API 配置：输入后端 API 地址，默认为 http://localhost:8000
  • 上传用户照片：上传你或模特的照片，建议选择清晰、光线良好的照片
  • 上传商品照片：支持上传单个或多个商品（如上衣+裙子+挎包），系统会智能识别商品类型并匹配合适的场景
  • 上传场景照片（可选）：如果上传了场景照片，所有4张图片都将符合此场景。不上传则由AI自动选择场景
  • 风格描述：简短描述风格和场景，AI会自动优化
  • 多商品组合：系统会自动分析每个商品的类型（服装、配饰、鞋类等），智能匹配组合方式和场景
  • 图片比例：生成的图片比例为9:16（1080x1920），适合社交媒体分享
  • 脸部细节：强化脸部细节刻画，确保五官清晰、皮肤质感真实、禁止面部视觉崩塌
  • 生成结果：一次性生成4张不同角度、不同表情、不同场景的营销图片
  • 启动后端：运行 python api_example.py 启动 API 服务
