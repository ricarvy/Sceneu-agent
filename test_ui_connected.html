📖 使用说明

  • API 配置：输入后端 API 地址，默认为 http://localhost:8000
  • 用户照片：上传你或模特的照片，建议选择姿态自然、光线清晰的原片
  • 商品照片：上传需要推广的商品照片，建议选择主体清晰、无遮挡的原图
  • 风格提示词：简单描述想要的风格，AI会自动优化成专业的生成描述
  • 启动后端：运行 python api_example.py 启动 API 服务
